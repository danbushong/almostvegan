# Almost Vegan
  ## Description
  Find good food with complex dietary restrictions
  ## Table of Contents
  * [Installation](#Installation)
  * [Usage](#Usage)
  * [License](#License)
  * [Questions](#Questions)
  ## Installation
  npm install
  <br>
  npm start
  ## Usage
  Find different recipe's if you are in the mood for a certain food. Recipe take in types of dietary restrictions and return content that is personalized to the user.
